# Программа поиска работы

Программа Python для поиска и управления вакансиями на hh.ru.

## Возможности

- Поиск вакансий на hh.ru
- Сохранение вакансий в JSON-файл
- Фильтрация вакансий по ключевым словам и диапазону зарплат
- Сортировка вакансий по зарплате
- Просмотр N лучших вакансий по зарплате
- Просмотр и фильтрация сохраненных вакансий

## Установка

1. Клонирование репозитория
2. Создание виртуальной среды:
```bash
python -m venv .venv
```
3. Активация виртуальной среды:
- Windows:
```bash
.venv\Scripts\activate
```
- Linux/Mac:
```bash
source .venv/bin/activate
```
4. Установка зависимостей:
```bash
pip install -r requirements.txt
```

## Использование

Запуск программы:
```bash
python main.py
```

Программа предоставляет следующие возможности:
1. Поиск вакансий
2. Просмотр сохраненных вакансий
3. Фильтр сохраненных вакансий
4. Выход

При поиске вакансий вы можете:
- Ввести поисковый запрос
- Указать количество верхних вакансий для отображения
- Фильтр по ключевым словам
- Фильтр по диапазону зарплат

## Структура проекта

- `src/` - Каталог исходного кода
  - `abstract_classes.py` - Абстрактные классы для API и хранилища
  - `hh_api.py` - Реализация API HeadHunter
  - `json_storage.py` - Реализация хранилища JSON
  - `vacancy.py` - Класс вакансии
- `tests/` - Тестовые файлы
- `data/` - Каталог для хранения данных о вакансиях
- `main.py` - Основной файл программы
- `requirements.txt` - Зависимости проекта

## Запуск тестов

Для запуска тесты:
```bash
python -m pytest tests/
```
